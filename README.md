# Chip8

## Building

### Build and run the standalone target

Use the following commands to build and run the executable target.

```bash
xmake -y
xmake run chip8 -h
```

## Dependencies

- C++17 compiler :
  - Visual Studio 2016
  - GCC 7+ or Clang 8+
- [XMake](https://xmake.io/) for build system creation (>= 3.12)
